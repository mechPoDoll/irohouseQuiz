# 🏠 이로하우스 카톡 퀴즈

이로하우스 단톡방의 대화 흐름을 보고, 빈 칸의 답변을 맞히는 퀴즈입니다.

## 퀴즈 구성

- 총 **10문제**, 매번 순서가 섞여서 나옵니다.
- 각 문제는 카카오톡 대화 화면처럼 보여집니다.
- 위에 **질문**(예: "똥집이 얼마였을까?")이 표시되고, 아래 대화창에서 정답이 들어갈 자리는 **???**로 가려져 있습니다.
- 답변 이전 메시지(앞맥락)는 흐리게 함께 보여서 추리할 수 있고, 답변 이후 메시지(뒷맥락)는 정답을 입력한 뒤에만 공개됩니다.
- 일부 문제(금액·시간 관련)는 너무 일찍 나오면 흐름이 깨지므로 항상 4번째 이후에 배치됩니다.

## 채점 방식

문제마다 정답으로 인정되는 표현이 다양합니다.

- **완전 일치**: 1점
- **부분 점수**: 답에 가까울수록 0.3 / 0.5 / 0.7 / 0.9점 차등 부여
- **만점 10점**
## 등급

- 90% 이상: 🏆 이로하우스 마스터
- 70% 이상: 🥇 거의 한 명의 멤버
- 50% 이상: 🥈 가끔 보는 친구
- 30% 이상: 🥉 인사 정도는 하는 사이
- 그 이하: 👀 처음 들어와봤어요

## 결과 화면

10문제를 다 풀면 총점과 등급이 표시되고, 그 아래에 모든 문제의 풀이 결과(정답/부분/오답 + 본인이 입력한 답)가 정리됩니다. "다시 풀기" 버튼으로 새 순서로 다시 시작할 수 있습니다.
